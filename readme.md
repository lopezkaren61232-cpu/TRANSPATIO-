# TRASPATIO

## Proyecto Transversal

TRASPATIO es un proyecto enfocado en el cuidado de las plantas,
la creación de espacios verdes y la promoción de prácticas
sustentables que contribuyan al bienestar de la comunidad.

### Objetivos

- Fomentar el cuidado de las plantas.
- Promover espacios verdes.
- Impulsar la educación ambiental.
- Desarrollar actividades de sostenibilidad.
